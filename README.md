# Time-Aware-CP-ABE-with-Threshold-Policies-for-Digital-Twin-Security-in-Industrial-IoT
T-CP-ABE v1.0 - Time-aware CP-ABE framework for Industrial IoT Digital Twins (IEEE IoT Journal submission). Features hash-chain time predicates, threshold policies, batch revocation, DDPM anomaly detection. Includes full source code, SOTA baselines, security tests, experiments, and paper LaTeX.
